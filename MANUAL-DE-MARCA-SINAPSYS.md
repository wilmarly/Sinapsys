## 1. Introducción

Este documento explica y muestra los aspectos que componen la identidad de marca de **Sinapsys**. Esta guía de marca funciona para:

- Mantener siempre la misma dirección
- Asegurar que la marca mantenga los mismos lineamientos y usos a lo largo del tiempo

## 2. Logos

### 2.1 Logo principal con neurona

- **Composición**: Dos elementos — el nombre de la marca y el símbolo
- **Uso de color**: El logo principal puede cambiar de color, siempre y cuando se usen las combinaciones propuestas en la paleta de color (ver sección 6)
- **Contexto**: Sobre fondo de color sólido
- **Restricciones**:
  - No se permite cambiar ni modificar el orden de los elementos
  - No se permite modificar su tamaño de forma desproporcionada

### 2.2 Logo principal

- **Composición**: Variación que solo se compone por el nombre de la marca (sin símbolo/neurona)
- **Uso**: Cuando se quiere usar con fondo en **degradé**
- **Uso de color**: Puede cambiar de color con las combinaciones propuestas
- **Restricciones**:
  - No se permite cambiar ni modificar el orden de los elementos
  - No se permite modificar su tamaño de forma desproporcionada

### 2.3 Logo secundario

- **Composición**: Nombre de la marca diagramado de manera más **vertical**
- **Contexto**: Sobre fondo en degradé
- **Uso de color**: Puede cambiar de color con las combinaciones propuestas
- **Restricciones**:
  - No se permite cambiar ni modificar el orden de los elementos
  - No se permite modificar su tamaño de forma desproporcionada

### 2.4 Símbolo

- **Significado**: Representación gráfica del movimiento que sucede en el cerebro cuando las neuronas generan la **sinapsis**
- **Función**: Gracias a su composición minimalista, sirve como abreviación del logo primario y del logo secundario
- **Uso de color**: Puede cambiar de color con las combinaciones propuestas
- **Restricciones**:
  - No se permite cambiar ni modificar el orden de los elementos
  - No se permite modificar su tamaño de forma desproporcionada

## 3. Área de reserva

El área de reserva protege a los logos de elementos gráficos para una correcta visualización. Elementos de los que debe protegerse:

- Imágenes
- Otros logos
- Textos
- Fotos
- Entre otros elementos gráficos

**Unidad de medida**: En todos los casos, **el ancho de la letra "S" de la palabra "Sinapsys"** determina el área de reserva.

| Elemento | Referencia de área de reserva |
|----------|-------------------------------|
| Logo principal con neurona | Ancho de la "S" de "Sinapsys" |
| Logo principal | Ancho de la "S" de "Sinapsys" |
| Logo secundario | Ancho de la "S" de "Sinapsys" |
| Símbolo | Ancho de la "S" de "Sinapsys" |

## 5. Usos incorrectos

Los siguientes usos están **prohibidos** para garantizar el impacto visual y la legibilidad. Aplican al logo principal con neurona, logo principal, logo secundario y símbolo:

| Prohibido | Descripción |
|-----------|-------------|
| **No alterar las proporciones** | Mantener la relación de aspecto original |
| **No usar varios colores para el logo completo** | El logo debe usar un solo color por aplicación |
| **No usar minúsculas** | Siempre mantener la capitalización definida |
| **No distorsionar** | No estirar, comprimir ni deformar |
| **No cambiar de ángulo** | No rotar ni inclinar |
| **No usar en delineado** | No usar versión outline/contorno |
| **No cambiar el orden de los elementos** | Mantener la disposición original (nombre + símbolo) |
| **No usar otros colores** | Solo colores de la paleta oficial |

## 6. Paleta de color

### 6.1 Principios generales

- La marca cuenta con **3 degradés** y una **paleta de 7 colores** (+ variantes de grises/negro)
- La combinación de colores **no debe comprometer la legibilidad** de los logos y el símbolo
- **CMYK**: Para impresión OFFSET en policromía (porcentajes indicados)
- **RGB y Hexadecimal (#)**: Para medios digitales

### 6.4 Los 3 degradés oficiales

La marca define **3 degradés** con paradas de color específicas. Deben usarse según el contexto para mantener coherencia visual.

#### Degradé 1: Azul — Azul oscuro — Morado
Flujo: azul brillante (inferior izquierda) → azul oscuro (centro) → morado (inferior derecha).

| Parada | HEX | RGB |
|--------|-----|-----|
| Inicio (inferior izquierda) | #297ABF | 41, 122, 191 |
| Centro (superior central) | #0E2651 | 14, 38, 81 |
| Final (inferior derecha) | #873CA0 | 135, 60, 160 |

```css
/* Dirección: izquierda (azul) → centro (oscuro) → derecha (morado) */
--grad-main: linear-gradient(90deg, #297ABF 0%, #0E2651 50%, #873CA0 100%);
```

#### Degradé 2: Verde oscuro — Verde — Amarillo
Flujo: verde oscuro (superior) → verde oliva → amarillo (esquinas).

| Parada | HEX | RGB |
|--------|-----|-----|
| Inicio (superior izquierda) | #263D0B | 38, 61, 11 |
| Centro (centro derecha) | #577019 | 87, 112, 25 |
| Final (inferior izquierda) | #F3CB04 | 243, 203, 4 |

```css
--grad-green: linear-gradient(135deg, #263D0B 0%, #577019 50%, #F3CB04 100%);
```

#### Degradé 3: Grises — Negro
Degradé sutil entre tonos oscuros para fondos y sofisticación.

| Parada | HEX | RGB |
|--------|-----|-----|
| Gris oscuro | #303030 | 48, 48, 48 |
| Negro | #000000 | 0, 0, 0 |
| Gris medio | #212121 | 33, 33, 33 |

```css
--grad-dark: linear-gradient(180deg, #212121 0%, #000000 50%, #303030 100%);
```

### 6.5 Resumen rápido para desarrollo digital (CSS/variables)

## 7. Tipografías

La marca usa una tipografía **sans-serif** para transmitir un mensaje moderno y cercano.

### 7.1 Títulos y mensajes cortos

| Tipo | Uso |
|------|-----|
| **Lato Bold** | Títulos y mensajes cortos |
| **Lato Black** | Títulos y mensajes cortos (mayor énfasis) |
| **Lato Bold Italic** | Para resaltar aún más |
| **Lato Black Italic** | Para resaltar aún más |

**Restricción**: Estas variantes **no están permitidas** para:
- Frases muy largas
- Párrafos
- Documentos enteros

**Caracteres de muestra**: ABCDEFGHIJKLMÑOPQRSTUVWXYZ | abcdefghijklmñopqrstuvwxyz | 1234567890 | °!"#$%&/()=?¡ | áü

### 7.2 Cuerpos de texto

| Tipo | Uso |
|------|-----|
| **Lato Light** | Frases, párrafos y documentos |
| **Lato Regular** | Frases, párrafos y documentos |
| **Lato Italic** | Para resaltar dentro de textos largos |

**Restricción**: Estas variantes **no están permitidas** para:
- Frases cortas
- Mensajes clave

**Caracteres de muestra**: ABCDEFGHIJKLMÑOPQRSTUVWXYZ | abcdefghijklmñopqrstuvwxyz | 1234567890 | °!"#$%&/()=?¡ | áü

### 7.3 Familia tipográfica completa

- **Lato** (Google Fonts)  
- Variantes usadas: Light, Regular, Italic, Bold, Bold Italic, Black, Black Italic

## Resumen de reglas clave para implementación

1. **Logos**: Usar las 4 variantes según contexto (fondo sólido vs degradé, horizontal vs vertical)
2. **Colores**: Solo paleta oficial; en digital usar HEX o RGB
3. **Tipografías**: Lato; Bold/Black para títulos, Light/Regular para cuerpo
4. **Área de reserva**: Basada en el ancho de la "S" de "Sinapsys"
5. **Tamaños mínimos**: 3 cm (logo con neurona), 2 cm (logos principales), 1 cm (símbolo)
6. **Usos incorrectos**: Evitar alteraciones de proporción, color, rotación o delineado
